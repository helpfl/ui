#  For SPA Helpfl

This is a spa for the helpfl project. 
It is a react app that uses the AWS CDK to deploy to AWS.

The site is hosted at https://helpfl.click

##  Prerequisites

* node js

## Useful Commands

Install dependencies:

```
npm ci
```

Start in watch mode:
```
npm run dev
```
Run tests: 
```
npm run test
```

Lint Code: 
```
npm run lint
```

Fix Linting Issues: 
```
npm run lint:fix
```

Build for production:
```
npm run build
```

See infrastructure diff:
```
npx cdk diff
```
